# Refranero Mix

> Mix aleatorio de refranes en castellano.


## Utilidades

Dispone de:

+ Una versión web.
+ Dos versión para usar en consola (una en Nim y otra en Python).
+ Un filtro/selector automatizado para tipos de refranes según su estructura (en Python).
